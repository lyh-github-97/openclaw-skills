# 微信公众号标题指南

## 核心原则

用户阅读标题时只有0.5秒决策时间。标题必须在：
1. **搜索友好** — 包含用户会搜索的关键词
2. **点击欲强** — 制造好奇心缺口或价值承诺
3. **系列感** — 形成持续阅读习惯
4. **去AI味** — 听起来像人写的，不是模板套的
5. **价值清晰** — 用户知道点进来能得到什么

---

## 评分维度说明

### 1. 搜索指数 (0-25)

用户通过订阅号列表或朋友圈分享看到标题。**关键词**决定是否被搜索到。

高价值关键词类型：
- **人物**：马斯克、特斯拉、库克、周鸿祎等名人
- **公司**：字节跳动、美团、京东、拼多多等
- **概念**：AI、第一性原理、自动驾驶、芯片
- **情绪**：暴雷、翻身、裁员、暴富

### 2. 点击欲 (0-25)

好奇心缺口是点击的核心驱动力。

高效触发词：
- **数字**：10个、3次、第1章、⑩
- **疑问**：为什么、怎么、是什么
- **冲突**：不是…而是…、翻车、暴雷、碾压
- **悬念**：不为人知、终于、第一次、睡不着
- **行动**：文末领、限时、免费

### 3. 系列感 (0-20)

系列文章是建立读者习惯的最强工具。

标准格式：
- `马斯克新书①` / `马斯克新书②` ...
- `埃隆新书①：xxx`
- 《书名》① ② ③
- `第一章 / 第二章`

系列标题一旦确定，**不要轻易改格式**，保持一致。

### 4. 去AI味 (0-15)

AI生成标题有明显的模板特征，去AI味 = 像人写的。

**必须避免的AI味模式：**
```
- "关于xxx的思考" / "对xxx的深度思考"
- "一文读懂xxx" / "一文看懂xxx"
- "深度好文" / "深度解读" / "深度分析"
- "干货满满" / "全是干货"
- "建议收藏" / "建议转发"
- "不容错过" / "不可错过"
- "字少事大" / "话不多说"
- "YYDS" / "绝了绝了"
- "太牛了" / "太强了"（过度感叹）
- "接下来" / "然后" / "接着"（废话衔接）
- "综上所述" / "总而言之"（AI总结腔）
- "具有深远意义" / "值得高度重视"
```

**自然表达替换参考：**
| AI味表达 | 自然替代 |
|---------|---------|
| 关于马斯克的深度思考 | 马斯克如何做决策 |
| 一文读懂第一性原理 | 第一性原理怎么用在工作里 |
| 深度好文建议收藏 | 读完少走三年弯路 |
| 干货满满不容错过 | 整理了做产品的核心方法 |

### 5. 价值感 (0-15)

用户凭什么点进来？标题要给出**明确的收益承诺**。

加分项：
- `文末领新书` / `免费领取xxx` — 明确福利
- 具体数字承诺 — `特斯拉的5个决策逻辑`
- 冲突反差 — `你以为xxx，其实xxx`
- 结果导向 — `差点死掉三次` / `从濒临破产到全球第一`

### 6. 长度合规 (0-5)

微信标题上限 **64个字符**（不是字数，是字符）。

- ≤64字：满分
- 65-74字：3分
- ≥75字：0分，可能被截断

---

## 标题优化工作流

1. **写完标题** → 跑 `score_title.py` 看评分
2. **低于60分** → 看问题列表针对性修改
3. **AI味超标** → 替换模板词，用人话重写
4. **搜索分低** → 替换或补充高搜索关键词
5. **系列文章** → 保持编号格式一致
6. **再次跑分** → 确认达到70+再使用

## 《埃隆之书》系列专属规范

适用于"万人荒野"公众号的《埃隆之书》系列：

**格式规范：**
- 前缀：`马斯克新书①：` 或 `马斯克新书①：`
- 编号用 ①②③④⑤⑥⑦⑧⑨⑩⑪⑫⑬⑭
- 文末钩子：`（文末领新书）`
- 单篇字数：25-32字为佳

**禁用词（AI味）：**
- "深度" "思考" "感悟" 不用
- "一文" 不用
- "干货" "收藏" "转发" 不用
- "YYDS" "太牛了" 不用

**合格标题示例：**
```
马斯克新书④：马斯克眼中真正的成功，不是赚钱，是这个（文末领新书）
马斯克新书⑧：第一性原理的终极实践：拆火箭，找最贵零件，然后让它变便宜（文末领新书）
```

**不合格标题示例：**
```
关于马斯克创业的深度思考与感悟  （AI味+无系列感+无点击钩子）
一文读懂马斯克新书的核心内容  （AI味模板）
马斯克新书④成功之道  （太平淡，无悬念，无福利钩子）
```
