# 标题评分器

评估和优化微信公众号标题的专属工具。

## 快速使用

```bash
python3 ~/.openclaw/skills/title-scorer/scripts/score_title.py "你的标题"
```

## 评分维度

| 维度 | 分值 | 说明 |
|------|------|------|
| 搜索指数 | 0-25 | 含有关键词数量和质量（马斯克、特斯拉、AI等） |
| 点击欲 | 0-25 | 好奇心驱动（数字、疑问、冲突、悬念） |
| 系列感 | 0-20 | 编号/章节标识，建立读者习惯 |
| 去AI味 | 0-15 | 避开模板套路（"一文读懂""深度好文"等） |
| 价值感 | 0-15 | 明确用户收益（文末领xxx、具体承诺） |
| 长度合规 | 0-5 | 64字以内 |

**总分70+为合格，80+为优秀，60-需优化。**

## 评分流程

1. 运行脚本获取评分和诊断
2. 重点关注 **去AI味** 和 **点击欲** 两项
3. AI味超标时，参考 `references/wechat-title-guide.md` 中的替换表
4. 修改后重新评分确认

## 快速优化清单

- [ ] 标题是否包含高搜索量关键词（人名/公司名/概念词）？
- [ ] 是否有数字编号或系列标识？
- [ ] 是否有好奇心触发（疑问/冲突/悬念/数字）？
- [ ] 是否有文末福利钩子（文末领新书/免费xxx）？
- [ ] 是否避开了AI味模式（一文读懂/深度好文/干货满满/建议收藏）？
- [ ] 是否在64字以内？
