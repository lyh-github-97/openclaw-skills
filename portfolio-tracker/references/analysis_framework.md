# 持仓分析框架

## 核心分析维度

### 1. 浮亏/浮盈分析
- 计算浮亏比例 = (现价 - 成本) / 成本 × 100%
- 浮亏 > 15% → 重点关注止损线
- 浮亏 8-15% → 观察是否破支撑
- 浮亏 < 8% → 持有待解套

### 2. 仓位分析
- 单只仓位占比 > 30% → 建议减仓分散风险
- 单一行业仓位 > 50% → 行业集中风险

### 3. 成本结构分析
- 重仓成本线：若现价低于成本20%以上，属于深度被套
- 轻仓成本线：若现价高于成本，属于盈利状态

### 4. 操作建议梯度

| 浮亏区间 | 仓位 | 建议 |
|---------|------|------|
| <-20% | 任何 | 止损或等反弹减仓 |
| -15% ~ -20% | >30% | 减仓至50% |
| -8% ~ -15% | 合理 | 持有，看支撑 |
| -5% ~ -8% | 合理 | 持有，等解套 |
| > -5% | 合理 | 解套目标明确 |

### 5. 消息面影响
- 个股黑天鹅（立案调查等）→ 止损优先
- 行业系统性下跌 → 等反弹减仓
- 个股独立行情 → 根据技术面决定

### 6. 时间维度
- 短期（1-2周）：看技术面和消息面
- 中期（1-3月）：看业绩和政策催化
- 长期（3月+）：看基本面和行业趋势
